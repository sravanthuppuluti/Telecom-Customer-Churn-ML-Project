# Telecom Customer Churn Prediction

## 📌 Project Overview
This project focuses on predicting customer churn in a telecom company using Machine Learning techniques. The goal is to identify customers likely to leave and provide actionable business insights. 

## 🎯 Objectives
- Predict customer churn
- Identify key factors influencing churn
- Provide business recommendations

## 📊 Dataset
- 2500 records
- 7 features + 1 target variable (Churn)
- Mix of categorical and numerical features

## ⚙️ Technologies Used
- Python (Pandas, NumPy, Matplotlib, Scikit-learn)
- SQL
- Machine Learning (Logistic Regression, Random Forest)

## 🔄 Data Preprocessing
- Label Encoding
- Standard Scaling
- Train-Test Split

## 🤖 Models Used
- Logistic Regression
- Random Forest

## 📈 Model Performance
- Accuracy: 75%
- Evaluation: Confusion Matrix, Precision, Recall, F1-score

## 🔍 Key Insights
- Poor internet service is the biggest factor affecting churn
- Customers leave due to service dissatisfaction

## 💡 Business Recommendations
- Improve service quality
- Provide retention offers
- Monitor customer complaints

## 📁 Project Files
- Documentation
- Presentation
- Model Output (HTML)

## 🚀 Conclusion
The model successfully predicts churn and provides insights for improving customer retention.
